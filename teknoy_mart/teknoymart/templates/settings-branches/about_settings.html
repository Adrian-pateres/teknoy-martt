Tech Stack

Teknoy Mart is built using a modern web stack chosen for reliability, maintainability, and ease of deployment.

  • Frontend: HTML, CSS, JavaScript (using Django Templates, with optional React integration).
  • Backend: Python, Django for business logic and security.
  • Database: MySQL and Supabase for structured data and secure storage.
  • Authentication: Django Authentication with university email validation to enforce @cit.edu access.
  • Deployment (optional): Heroku, Vercel, or Docker-based setups for cloud hosting and scalability.
