Platform Overview

The settings section of the CIT-U eCommerce Platform (Teknoy Mart) lets students personalize how they use the system, manage their privacy, and understand how the platform works. These tools are designed to keep transactions secure, convenient, and aligned with university policies.

Mission Statement

To provide a safe, convenient, and trusted marketplace exclusively for CIT-U students to buy, sell, and exchange items within the campus community.

Tech Stack

Teknoy Mart is built using a modern web stack chosen for reliability, maintainability, and ease of deployment.

Frontend

Client-Side

  • HTML5 - Semantic markup
  • CSS3 - Modern styling & animations
  • JavaScript - Interactive features
  • Django Templates - Server-side rendering
Backend

Server-Side

  • Python 3.13 - Core language
  • Django 5.2 - Web framework
  • Django REST Framework - API development
  • Celery - Asynchronous tasks
Database

Data Storage

  • PostgreSQL - Primary database
  • Supabase - Cloud-hosted PostgreSQL
  • Redis - Caching & sessions
  • Cloudinary - Media storage
Security

Authentication

  • Django Auth - User management
  • Email Verification - @cit.edu validation
  • CSRF Protection - Form security
  • SSL/TLS - Encrypted connections
Deployment

Hosting

  • Render - Primary hosting
  • Vercel - Alternative hosting
  • Docker - Containerization
  • GitHub Actions - CI/CD pipeline

How Students Use Teknoy Mart

Students log in with their CIT-U email, configure their preferences and privacy settings, then browse, post, or manage listings from any device with internet access. Messaging tools and secure authentication help keep every transaction organized and safe within the campus community.

1

Register with CIT-U Email

Sign up using your official @cit.edu email address. Verify your account through the confirmation email.

2

Complete Your Profile

Add your details, profile picture, and preferences to personalize your marketplace experience.

3

Browse or List Items

Search for items you need or post your own listings with photos, descriptions, and pricing.

4

Connect & Transact

Use the built-in messaging system to communicate with buyers or sellers and arrange meetups.

Safe & Secure

All transactions happen within the CIT-U community. Meet in public campus locations and use the platform's messaging system to protect your privacy.
