# CIT-U eCommerce Platform

An exclusive online marketplace for **Cebu Institute of Technology - University (CIT-U)** students.  
This platform allows students to **buy, sell, and rent** items within the campus community.  
To ensure security, only users with a **valid CIT-U email address** can register and log in.
